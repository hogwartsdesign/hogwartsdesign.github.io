About Hogwarts Design Changes

Welcome to Hogwarts Design Changes, my new website! I decided to create this website after researching various changes to the design of Hogwarts Castle in each of the Harry Potter movies for fun.

Each page on this site contains the changes from each Harry Potter movie. I left out pages for Philosopher's Stone and Deathly Hallows - Part I because Philosopher's Stone has no changes (as it is the first movie) and Hogwarts only appears as flashbacks in Deathly Hallows - Part I.
